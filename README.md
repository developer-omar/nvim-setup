# My Nvim Configuration

## Requirements

1. Install luarocks for compile some plugins

    ```bash
    sudo apt install luarocks
    ```
2. Install ripgrep

    ```bash
    sudo apt install ripgrep
    ```
3. Install fd-find

    ```bash
    sudo apt install fd-find
    ```
4. Install fzf

    ```bash
    sudo apt install fzf
    ```
